# Integrating-ICM-42688-P-and-LIS3MDL-together-through-sensor-fusion-on-STM32H743ZIT6
This Repository includes all the codes and Necessary files for integrating this specific IMU (ICM-42688-P) and Magnetometer  (LIS3MDL) on STM32H743ZIT6

To calibrate these sensors and find the initial offset of all 3 axis by using the commented out code in main.c and rotate the board for 30 seconds and then put the calculated offset in the code .
